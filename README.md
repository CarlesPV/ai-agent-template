# Proyecto Base para Agentes de IA

Plantilla estructural optimizada para el desarrollo de software asistido por Inteligencia Artificial y agentes autónomos (Antigravity, etc.).

## Estructura del Proyecto
- `/docs`: Contexto y guías para el agente de IA.
  - `/requirements`: Definición de lo que se va a construir.
  - `/documentation`: Manuales y referencias técnicas.
  - `/history`: Registro de cambios, comandos ejecutados y decisiones tomadas por el agente.
- `/src`: Código fuente de la aplicación.
- `/tests`: Pruebas unitarias y de integración.

## Cómo empezar
1. Clona este repositorio.
2. Copia el archivo `.env.example` a `.env` y añade tus variables locales.
3. Instruye a tu agente para que lea primero la carpeta `/docs/requirements`.
4. Reemplaza el contenido de este README con la información real de tu aplicación una vez comiences el desarrollo.
